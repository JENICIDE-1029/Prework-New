HTML

  • HTML is what builds the framing to our website
  • The head element is not visible to users, considered a parent, and it contains information about the page for the browser
  • The body element is considered a parent and contains what end-users are able to see

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When using the external css method, which involves a seperate file that is referenced within the HTML file (within the head, after meta)

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: save changes to repo

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • With control flow in mind, it is important to identify when variables are defined and when they are being used to ensure they are called upon only after they've been defined
  • Functions help us to not only re-use code but grant us more control in if that code is even used
